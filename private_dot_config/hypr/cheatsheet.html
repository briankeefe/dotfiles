Hyprland Cheat Sheet

The rule: ctrl-super = window manager. alt = workspace numbers. Same mental model as the AeroSpace config.
Homes: 1 Ghostty · 2 Slack · 3 DataGrip · 4 Chrome · 5 junk drawer

Summon / banish

ctrl-super+gGhostty home 1
ctrl-super+sSlack home 2
ctrl-super+dDataGrip home 3
ctrl-super+cChrome home 4
ctrl-super+rReset apps home panic
Ghostty→1, Slack→2, DataGrip→3, Chrome→4; strays→5

Workspaces

alt+1-5Switch workspace
alt+shift+1-5Move focused window to workspace
homes1 Ghostty · 2 Slack · 3 DataGrip · 4 Chrome · 5 junk

Window control

ctrl-super+h/j/k/lFocus left/down/up/right
ctrl-super+shift+h/j/k/lMove window left/down/up/right
ctrl-super+mFullscreen
alt+shift+spaceToggle floating focused window

Workspace modes

ctrl-super+shift+fFloat current workspace
ctrl-super+shift+tRe-tile current workspace
ctrl-super+returnOpen Ghostty
ctrl-super+spaceOpen app launcher

Notes

●Cyan border tracks the focused window.
●New app windows auto-route home by class. Confirm classes with hyprctl clients if a new Linux install differs.
●Summon toggles are scripts under ~/.config/hypr/scripts/; no giant inline shell in the config.
●Free for terminal/herdr/vim: bare alt-h/j/k/l, alt-f, and cmd-less letters.
Config: ~/.config/hypr/hyprland.conf · Reload: hyprctl reload Linux-only via chezmoi · mirrors AeroSpace homes and summon behavior

AeroSpace Cheat Sheet

The rule: cmd-ctrl = window manager (everything). alt + digit = workspaces.
Mental model  ·  cmd-ctrl-h/l moves the cyan highlight between tiles (focus only). Add shift and the focused tile physically trades places with its neighbor.

Windows → cmd-ctrl

cmd-ctrl+hjkl Focus left / down / up / right — moves the cyan highlight
cmd-ctrl+shift+hjkl Move focused window — swaps tiles
cmd-ctrl+m Fullscreen toggle (maximize)
cmd-ctrl+g Summon Ghostty to current workspace
cmd-ctrl+c Summon Chrome
cmd-ctrl+s Summon Slack
cmd-ctrl+d Summon DataGrip
cmd-ctrl+r Reset to halves panic button
Ghostty+Chrome→WS1, DataGrip+Slack→WS2, sweep extras→WS3
cmd-ctrl+shift+f Float whole desktop (free-drag mode)
cmd-ctrl+shift+t Re-tile whole desktop

Workspaces → alt + digit

alt+1234 Switch to workspace N
alt+shift+1234 Banish focused window to WS N

Layout odds & ends → alt

alt+/ Force horizontal columns
alt+, Accordion layout
alt+shift+space Float/tile just the focused window
